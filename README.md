## Design Decisions

- **Clean and functional** — no hand-drawn flourishes or artistic flair
- **Forest green palette** — `#4a6741` primary, natural tones throughout
- **Information-dense** — hours, phone, address always visible in header
- **Modeled after authentic local restaurants** — Golden Fleece, Clark Lake, Pegasus Taverna style
- **NOT a landing page** — functional restaurant site that shows real info upfront
